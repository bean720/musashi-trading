# Musashi Trading System

This repository contains the base files for the Musashi Trading System - a comprehensive MT4 trading framework.

## Contents

### 1. ExitEA_v1.1
- **ExitEA_v1.1.mq4**: Main Expert Advisor source code
- **ExitEA_v1.1.ex4**: Compiled Expert Advisor
- **Signals/**: Signal handling modules
- **MQH/**: Include files (structs, enums, trading manager)
- **Resources/**: Additional indicators and resources

### 2. Musashi_EA_v1.8.5
- Main Musashi Expert Advisor version 1.8.5
- Complete trading system with entry/exit logic
- Configuration files and documentation

### 3. TF Action & Exhaustion Indicator v10.2.3
- Timeframe action and exhaustion indicator
- Divergence/convergence detection
- Arrow formation logic
- Sorting algorithms for market analysis

## File Types

- **.mq4**: MetaTrader 4 source code files
- **.ex4**: Compiled MetaTrader 4 executable files
- **.mqh**: MetaTrader 4 include/header files
- **.zip**: Archived versions for distribution

## Purpose

This is the foundational codebase for the Musashi trading system. These files represent the base version that will be extended and improved upon for live trading operations.

## Usage

1. Copy the `.ex4` files to your MT4 `Experts/` or `Indicators/` directory
2. Source files (`.mq4`, `.mqh`) can be modified and recompiled in MetaEditor
3. Refer to individual component documentation for configuration details
